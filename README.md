# `guoshaoyang-pku.github.io`

Personal site for Shaoyang Guo, built with Astro and designed for GitHub Pages.

## What is included

- A one-page home page focused on positioning, selected work, experience, and contact
- A public-facing `CV Summary` page
- GitHub Actions workflow for Pages deployment
- Content centralized in `src/data/site.js` for easy editing

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize next

- Replace the temporary private-project notes with public repo links when they are ready
- Add a downloadable PDF CV if you want a stronger recruiting-oriented CTA
- Add short notes under `Writing / Notes`
- Configure a custom domain later if needed
